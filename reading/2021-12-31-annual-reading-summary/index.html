2021年读书流水账

如果要选择一个年度关键字的话，我会毫不犹豫选择“累”，真的很累。

但非常意外今年的阅读量却创新高，主要原因是重度使用微信读书，在抱小宝睡觉期间增加大量阅读量。

最终豆瓣标记阅读量为114本，当然通过本数来衡量阅读量并不客观，比如这里面有一些育儿书就是陪孩子时一中午翻完的。另外还有不少读库小册子，一百多页的小册子基本上一到两天通勤就能看完。整个12月上下班读完了《医学大神》14本。当然也不乏一些社科大部头，比如《金与铁》看了断断续续几个月，M译丛的《第三帝国三部曲》也是肝了几个月。互相平衡下来也不算太不客观。

输入量是多了，但输出却少得可怜，这也是年度关键字“累”的原因。阅读都是通勤、陪孩子、外出办事排队、等电梯等等碎片时间，能坐在书桌前的时间却几乎没有，或者有时间的时候身体已被琐事掏空只想躺平，还要抽空玩游戏。甘蔗没有两头甜，看书多了学习就少了。事已至此也只能安慰自己读这些书带来很多快乐，仅此而已。甚至这篇流水账也是持续了很久才难产出来的。

旅行文学

这两年因为疫情旅行成为一件很困难的时期，当然这对我来说问题并不在这里，我的问题一直是没钱。虽身不能至，心仍向往之，一直以来对世界各地的风俗人情都很感兴趣。借着作者的眼睛去看世界也不失为一种视角。

  • 失落的卫星 : 深入中亚大陆的旅程
  • 沿着季风的方向 : 从印度到东南亚的旅程
  • 午夜降临前抵达

年初从大火的《失落的卫星》开始，陆陆续续读完了刘子超的另外两本书，关于东南亚和南欧的 《沿着季风的方向》和《午夜降临前抵达》。作者深入观察当地人的生活，结合历史变迁政治影响，呈现一个个国家城市的人文地貌社会形态。但有时候总觉得浅尝辄止，略显单薄。另外在读书群有读者对作者过于直男优越感感到不适。

  • 騎乘鐵公雞 : 搭火車橫越中國
  • 火车大巴扎 : 横贯欧亚的火车之旅
  • 老巴塔哥尼亚快车
  • 英国环岛之旅
  • 赫拉克勒斯之柱

去年感觉身边无人不在谈论保罗索鲁，因为后浪出了他一本书《在中国的大地上》，之前在微博看到曾和他一起吃过饭的大佬评价其是一个渣、傻叉……因此更激起了想看的欲望。无意中找到了这本台版《騎乘鐵公雞》，看完感受到了保罗索鲁的刻薄，但从我个人来说倒不太在意，反而作为80尾的我认为其很好的还原了一个出生前的中国图景，有贾樟柯电影的画面感。而且他的一些刻薄看法到今天也依然没有太大的进步，比如国人爱吐痰，好面子。不过城市的脏乱差倒有很大的进步。最后一章更是看到了在其他地方看不到的一些观点和历史的另一面。另外几本《火车大巴扎》《老巴塔哥尼亚快车》《英国环岛之旅》《赫拉克勒斯之柱》，关于横跨欧亚、穿越南美、环岛英国、绕地中海的游记一样是毒舌且精彩。

  • 从大都到上都 : 在古道上重新发现中国

在这个系列中个人最佳当属罗新老师这本《从大都到上都》。颇有深度，甚至说田野考察笔记也不为过。罗老师不愧是历史研究者具备渊博的知识水平和优秀的文字功底。看到网上的评价“山川带字幕”。

  • 重走 : 在公路、河流和驿道上寻找西南联大

    《重走》在年底的好几个好书榜中都看到了它的身影，个人也非常喜欢这段重走1938年由清华北大南开组成的长沙临时大学徒步西迁昆明旅程。两段时空交织，时而是当下时而是80年前，蒙太奇的阅读体验。在车上读到林徽因和航空学院学员那段泪流满面。

人物 其实比较少看人物传记，因为一向认为很难有客观的人物传记，不管是自传还是他人所著。

  • 夜航西飞

    作者1936年为第一位女性独自驾机从英国飞越大西洋直抵北美，这本书也是记录作者硬核的人生经历，出生英国长在非洲，人生两大爱好，训练赛马和开飞机，性格自由彪悍。这种传奇人生特别适合拍成电影，而演员则非弗兰西斯·麦克多蒙莫属。

  • 上学记

  • 许渊冲：永远的西南联大

    5月份有一次在微博时间线看到何兆武先生去世，于是赶紧找到这本《上学记》来看。还没看又看到许渊冲先生去世的新闻，于是又接着看了这本《永远的西南联大》。一本是口述一本是自传，听老先生娓娓道来讲过去的故事。

  • 我的几何人生 : 丘成桐自传 看这本书可以吃到很多数学学术界的瓜，但不管丘先生是否客观，丘先生在数学学术界的成就和影响力确实是华人第一人。

  • 岩田先生 : 任天堂传奇社长如是说 作为游戏从业者，如果算的话。在这本《岩田先生》一上微信读书就去看了，不过书一般，一些访谈文章合集而已。

电影 虽然今年没怎么看电影，看着一些好电影没时间看也着急。不过看了几本与电影有关的书。

  • 三十年细说从头 挺厚一本书，100多万字，来自于李翰祥连载的专栏。几百字一篇特别适合碎片时间。基本是抱娃睡觉、等电梯的时间看完。讲述李翰祥先生经历的香港电影历程以及一些北京民间文化。语言风趣口语化时常忍不住爆笑。喜欢港片的朋友不容错过。

  • 十年一觉电影梦 : 李安传 曾经也自诩为电影爱好者，前几年也刷过不少电影，如果要我选一个最喜欢的电影镜头，不假思索会选择《饮食男女》开头郎老爷子做饭那个长镜头，看完这本书好想把李安的所有电影再看一遍。

  • 香港电影的秘密 : 娱乐的艺术 作为看香港电影长大的一代人，看得都是热闹，而通过这本《香港电影的秘密》则可了解其中的门道。一个歪果仁能通过电影延伸到当地文化、电影工业化体系的形成以及电影本身的风格发展，实在令人佩服。

虚构 小说看得比较少，感觉自己的情感并不能从小说中得到反馈，往往就是看了一个故事而已。

  • 南货店 半本好书，前后两部分差距太大，前半部的人物设计故事设定方言都特别好，但后半部就纯官场小说，只剩下乱搞男女关系了。但是只冲着前半本也值得看。

  • 两京十五日 亲王的书一直都挺适合影视化，这部很明显就是冲着影视剧改编的。结果没想到后面的《洛阳》《长安的荔枝》直接就写剧本了。

  • 金阁寺 三岛热那阵跟风看的，日本战后一代的作品普遍阴郁，喜欢不起来。

  • 永隔一江水 和过往的作品一样依然是那个邓烷，依然是那个方言。每每读邓安庆的作品都有一种亲切感，他的“邓烷”和我的“李家”都是一样的南方农村一样的人情世故。

  • 拳

  • 春山 何大草老师这两本书是吃群友安利，《拳》挺有意思的回忆录式的小说，故事其实挺散的，但是感觉就特别有成都味，安逸，喝茶摆龙门阵的感觉。但《春山》不是我的菜。

历史

  • 大屠杀 : 一部新的历史

  • 奥斯维辛 : 一部历史 这两本并行看的，从副标题可以看得出两本书讲的是同一件事，就是屠杀犹太人。除了广为人知的奥斯维辛还有贝乌热茨、索比堡等集中营、屠杀场。不止德国，甚至大半个欧洲都存在反犹行为，一个意外是意大利居然一定程度上保护过一些犹太人。

  • 埃及的革命考古學 : 在埃及挖掘被埋葬的革命和歷史 何伟去4月份左右年在纽约客发表关于新冠那篇文章的时候，群友说何伟运气太好了，每次都能在“大历史”中心。对于本书最好看的还是普通人的生活描写，收垃圾的人、同性恋记者、卖内衣的中国商人…对于革命、政治反倒没那么有兴趣。虽然书中写的内容是十来年前的事情，但仍然震惊于21世纪居然还存在女性割礼这种愚昧的事情，埃及女性的生存可能比大清国还艰难。最后关于何伟全家在埃及所居住的蜘蛛网大楼历史挺让人泪目的，以及带领扎伊尔德全家去金字塔参观遭到警察盘问，最后却为他全家免票并且感谢他为这座城市付出也令人感动，全书看下来终于看到这个国家温暖的一面。

  • 王氏之死 : 大历史背后的小人物命运 经典的通过普通人的人生经历作为历史观察窗口的学术著作。年末作者去世了，R.I.P。

  • 间谍与叛徒 : 改变历史的英苏谍战 之前在盖茨的书单中看到这本就很感兴趣，居然这么快就有简中版。非常精彩的故事，但居然还是真事。估计很快也会被拍成电影吧。

  • 向您告知，明天我们一家就要被杀 : 卢旺达大屠杀纪事 看过《卢旺达大饭店》，看这本书时依然令人窒息。到底是什么原因会让人如此凶残，视同类甚至熟人、亲戚邻居、师生为草芥随意砍杀。可怕，太可怕。

  • 拉丁美洲被切开的血管 说到拉美，想到的词语有“新大陆”“殖民掠夺”“热情奔放”“军政独裁”“魔幻现实主义”。。。本书是一位拉美作家对资本主义的控诉。矿产、土地、种植园，西班牙、葡萄牙、英国、美国轮番登场从这片土地大肆掠夺大量资源，成为一个个帝国。然而拉美本土却依然贫困混乱依然“魔幻现实主义”。

  • 第三帝国的到来 : 第三帝国三部曲01

  • 当权的第三帝国 : 第三帝国三部曲02

  • 战时的第三帝国 : 第三帝国三部曲03 这套书共花了24.5小时，真是“巨著”，很难用语言去概括在读书过程中的感受，但会迸发出这样一些词语，狂热、无脑、造神、洗脑、愤怒、失去理智、极端、杀戮、战争、惨烈、疾病、冷血、乌合之众、种族主义、民主主义、独裁主义…… 无论是男人女人儿童老人，无论是犹太人吉卜赛人斯拉夫人波兰人，无论是战俘小偷异己者异教徒，只要能找到理由格杀勿论。枪杀、饿死、活活打死、疾病以及流水线的毒死，只有想不到没有纳粹做不到。整个过程咬牙切齿悲愤交加，甚至到最后第三帝国灭亡这些罄竹难书的刽子手们绝大部分也没有收到应有的裁决。总之这套书值得任何人阅读思考。

  • 金与铁 : 俾斯麦、布莱希罗德与德意志帝国的建立 这本书挺难读的，而且又极其长，断断续续花了有3个月，以至于持续几个月看完以后几乎忘记所有内容。

  • 八月炮火 一战入门书，写一战起因，两大阵营争取联盟的角逐以及早期的几场大战。其实不算很好读，可能是因为写得很详细，所以涉及细节地名人名比较多，反正我都记不住。

  • 切尔诺贝利的午夜 看美剧《切尔诺贝利》之前应该看看这本书，非常好的补充。这本书没有任何意识形态的倾向，只是客观描述当年的事实。

  • 黑旗 : ISIS的崛起 isis组织发展的历史，感觉中东那地方的极端组织跟新冠病毒似的，干掉一个来一个，一个比一个猛。还是李光耀说得有道理，在石油经济被取代之前，ysl的问题基本上很难。

  • 季风吹拂的土地 : 现代东南亚的碎裂与重生 因为公司所面向的业务主要是东南亚，主动想去了解东南亚文化，个人特别喜欢这本书。东南亚一直以来充满了各种冲突，文化、宗教、种族，西方的民主和东方的儒家思想都没能在这生根发芽站稳脚跟。作者深入东南亚40年对此有深刻地见解。

  • 无规则游戏 : 阿富汗屡被中断的历史 在微博看到一个调侃“时令书“，读这本书其实也算得上”时令“，以为塔利班夺得政权所有媒体都在讨论塔利班而跟风去找来看的。本书内容包括从阿富汗建国到击毙本拉登的历史，正如书中的普什图谚语“我和我的兄弟一起对付我的堂兄弟；我和我的堂兄弟一起对付外人；当没有外人时，我对付我的兄弟”。这块土地也是一直各种部落政权打来打去，曾经有过非常世俗现代的时光，然而现在又进入万劫不复境地。

  • 圣经与利剑 : 英国和巴勒斯坦 : 从青铜时代到贝尔福宣言

  • 以色列 : 一个民族的重生

  • 我的应许之地 : 以色列的荣耀与悲情

  • 阿拉伯的劳伦斯 : 战争、谎言、帝国愚行与现代中东的形成 以上几本其实算是一个主题阅读，最早读《圣经与利剑》非常吃力，忍了半本最终还是弃了。于是有了后面两本比较通俗的进行入门，以前一直不明白，几百万人就这样任纳粹屠杀，纵然钢铁德国武器要强大很多但也可以打游击小规模抵抗啊，反正横竖一死。看完这本书才了解犹太人内部也有不同派系、由一群不同背景的人组成，可以说“犹太人”只是他们唯一的共同点。即使是经过万重困难建成的以色列也是矛盾重重。《阿拉伯的劳伦斯》则是好几年就想看，但一直没有相关知识储备而没敢翻开的书。这次趁热打铁也看了。欧洲帝国主义给世界留下了多少烂摊子，除了中东，整个亚非拉很多民族矛盾根源都可以追溯到那些欧洲老牌帝国主义随意瓜分划线所致。而英国就是那根最大的搅屎棍。本书从英国德国美国的情报人员即劳伦斯普吕佛耶鲁多视角叙事，结合他们背后的政治势力和个人经历来追溯今天中东格局形成的来龙去脉。看完这几本书算是对中东的近代历史脉络有点大致了解。

  • 什么也别说 : 一桩北爱尔兰谋杀案 这又是一本“科普书”，非常精彩，特别符合我的口味。以前听过一个说法伦敦街头没有垃圾桶或者说垃圾桶是透明的，其原因是防止共和党放炸弹。看了这本书才对共和党有一个基本认识。

  • 战争的试炼 : 十字军东征史 前两年就对中世纪十字军感兴趣，陆陆续续读了几本相关的书。到目前为止，这本是十字军历史最好的一本。荡气回肠，热血沸腾，主要写了第一次和第三次十字军东征，狮心王理查和萨拉丁的对决真是让人拍案叫绝永远是让人百看不厌的故事。

  • 条顿骑士团 : 一部军事史 可能是国内关于条顿骑士团相关的书太少，导致豆瓣分数虚高。

其他非虚构

  • 大地上的亲人 : 一个农村儿媳眼中的乡村图景 多年前曾经被一篇《一个乡村儿媳眼中的乡村图景》刷屏。当时看完并没有太大感触，本书则是基于那篇文章扩充，黄灯老师对婆家、自己家以及自己成长所处的外婆家进行更深入的剖析记录。书中记录的是湖北和湖南的农村，虽然三个乡村之间相隔几百公里，却能隔空呼应，在我眼里甚至能和我所处的江西农村一样能呼应起来，他他们有一个共同的名字“农民”，八亿分之一。这几十年来农村在经济结构、人情世故、从业手段、风俗道德的变迁可以说超过过去几百年。这些巨大的变迁也带来人们生活的震荡，或变好或变差。不管是好坏，只是时代的一粒灰。对于黄灯老师的书，包括前一本《我的二本学生》。给我的触动主要来自身份认知。通过读书离开农村，但亲人们都还在农村。农村早已不是我们童年认知中的家乡。农村的亲人我们熟悉又不熟悉。

  • 梁庄十年 至此“梁庄三部曲”看完，虽然前两本内容早已忘记。同样是农村题材相对于《大地的亲人》写的南方农村，梁庄对我来说多了一份陌生和猎奇。尤其是第二章写上一代女性的遭遇，令人震惊。细细想来书中几个梁庄姑娘所遭遇的人生，其实在家乡也是很平常的存在，正是因为它太过于平常反而从来想过去观察。

  • 东京绮梦 : 日本最后的前卫年代 别的不说，日本这个前卫艺术可真让我开眼界了。

  • 奶酪与蛆虫 : 一个16世纪磨坊主的宇宙 看标题可能会认为这是一本美食书，但实际是宗教题材。本以为这个题材会很晦涩，但实际上很好读，故事很好语言流畅。当权者想要奴役人首先就要使人思想麻痹愚蠢。这个磨坊主在中世纪竟然敢有思想，脑后反骨必然被审判。

  • 桶川跟踪狂杀人事件 特别适合在当下网暴事件频出的时代背景下读，这种事件往往间接伤害远大于直接伤害。不去谈社会或者公权力应该怎么去阻止这种恶性事件的发生。作为个体，直接用网络用语“做个人吧”，永远不要在网络上去伤害任何一个人。这是任何人都可以做得到的，也是网民的最基本道德吧。

  • 活着回来的男人 : 一个普通日本兵的二战及战后生命史 读完让人感慨万千，相对于现在网络上那些生活离不开宏大叙事的人，这种在时代洪流中飘零的人生才是真实的人生。作为个体，只有抱着“只要还有希望人就能活下去。”这样的信念活下去才能对抗这世界的种种。

  • 张医生与王医生 最近几年东北话题总是能引起我的兴趣，比如贾行家比如这两年的东北文艺复兴。这本讲的是两个东北医生的阶级跃迁故事，一个人的命运呐，自我奋斗和历史进程一个都不能少啊。

  • 下沉年代 七十年代至今美国社会全景图，好看是好看，但对美国政治完全不了解，故事之外的意义还是看不懂的。感觉又开了一个坑，但这就是读书的意义啊，时刻保持好奇心。

  • 深时之旅 我愿称之为“旅行文学”天花板，不过这本书分类还是属于新知科学，地下矿井中的暗物质实验室、真菌组成的树维网、石器时代洞穴中寻找人类过去的影子、无星河的源头、在北欧的冰块中分析过去的天气变化、探访核废料掩埋井…一个个硬核的科学概念一次次奇幻的旅程。加上作者优美的文笔读起来有点迷幻感。

科普 回头来看，今年看的科普书基本上都和医学相关，疫情时代，尊重科学尊重现代医学显得更为重要。这些书有些好读有些涉及到专业名词不太好读。

  • 消失的微生物 : 滥用抗生素引发的健康危机 抗生素无疑是最伟大的医学发现之一，拯救了无数生命。但近年来滥用抗生素也带来了很多负面的影响，尤其是有了孩子之后对一些儿科医生随便开抗生素很是无奈。看完此书发现这个担忧其实没有意义，因为从其他地方滥用的更多。

  • 仿制药的真相 非常精彩，拿起来就不想放下。《我不是药神》让我们知道了“仿制药”，看起来这是穷人的救世主。但是正是如此其中利益巨大，在利益面前人性总是那么经受不住考验，各种手段触目惊心。本书情节跌宕起伏比小说还是刺激，拍成电影一定很精彩。

  • 最好的告别 : 关于衰老与死亡，你必须知道的常识

  • 医生的修炼 : 在不完美中探索行医的真相

  • 医生的精进 : 从仁心仁术到追求卓越 看完阿图的这套三部曲，更激起了我对医生的崇拜，上一次还是18年自己做完手术，当时让自己近乎崩溃的健康问题在医生、现代医学的治疗下恢复，崇拜之心油然而生。作为中年人的我虽然能感受到身体的老化但实际上离衰老还是有一段距离。可是我们的父母却真的老了，养老、老年疾病、护理这些是摆在我们眼前的事情，不分贫富。本书通过一个个鲜活的故事来说明怎么刚好面对衰老面对死亡，可是这是美国的情况，而我们的医疗环境养老环境显然远不如，所以我们面临的情况更严峻。

  • UNIX传奇 : 历史与回忆 如果没有unix就没有今天的软件、操作系统、互联网，这本书介绍了unix是怎么来的，里面出现的每一个名字都值得顶礼膜拜，他们都是人类信息革命路上最亮的星。

  • 抗生素的故事 : 一颗改变人类命运的药丸

  • 耐药菌小史 : 一场抗生素与病原体之间的漫长斗争 两本关于抗生素的书，主题却略有不同。磺胺药、青霉素的发现与提炼挽救了无数人的生命，但抗生素绝不是圣杯，现在人类正在和耐药菌进行一场军备竞赛，看书过程中想到自己及家人使用抗生素的经历，以及畜牧业使用抗生素的量，触目惊心啊。在《抗生素的故事》中，可以了解到药品的发展除了科学家和医生们孜孜不倦的研究实验，那些因为药品不成熟而付出健康甚至生命的用药者也是背后的英雄。他们是一个更大的群体，也说最可怜的一群人。

  • 优雅的守卫者 : 人类免疫系统的故事 这本书其实是通过4个免疫系统疾病患者与病魔抗争的故事，来科普免疫系统的工作原理和医学的发展历程。本书其实不算科普书，上架类别也是非虚构，不过里面也有不少专业名词，对我来说也算科普。

  • 看不见的敌人 : 病毒的自然史 这是一本关于病毒的科普书，非专业人士可能很难分清楚病毒和细菌是两码事，对于常见疾病甚至大流行的病是由细菌还是病毒引起的。

  • 人类大瘟疫 : 一个世纪以来的全球性流行病 过去一个世纪大流行病的科普作品，黑死病、鼠疫、鹦鹉热、军团病、艾滋病、SARS、埃博拉、寨卡。。。这些病在爆发时期无一不是来势汹汹，但现在基本上可以说已经在人类的控制范围之内了。而我们当下正面临一个前所未有的新冠病毒，刚好今天看到南非又有一个更猛的变种。疫情什么时候结束真是遥遥无期了。

  • 为人文教育辩护 我们从小的教育是“学好数理化走遍天下都不怕”人文教育往往是被忽视的，课外书被氛围“有用的”和“没用的”，人文教育往往是属于“没用的”。作为理工科生，从工作中观察发现人文教育优者往往能把路走得更宽，比如更优秀的写作能力、表达能力、自我运营能力、对工作生活的观察能力。是时候改变观点了。

  • 棋局 虽然不懂围棋也不了解围棋江湖的历史恩怨，更看不懂围棋的招式。但从文字中依然能感受到江湖肃杀凶险的门派斗争气息。

  • 超文本和超链接 互联网在今天几乎是每一个人每一天都生活在其中，点开一个新闻页面、点开小程序点餐买票等等，都有http协议的支撑和html的渲染。但它的发展故事却鲜为人知。其实作为程序员学习的都是协议本身的内容，从未了解过其背后的故事，它的灵感来源、历史变迁。感谢这本小册子带我们走进历史，还原它的发展过程。

  • 达尔文发现了什么 : 今视角《物种起源》 重构精简版《物种起源》，虽然小小一本但可以通过这本小册子了解达尔文的思想，联想到达尔文可是大清朝时代的人，不禁感叹“达尔文可真是个有钱的天才啊”。

  • 地球上最伟大的一场演出 大概十来年前也不知道在网络哪个角落知道了live aid这场演出，把演出的视频下载下来，还拷贝一份在移动硬盘。反反复复听了一遍有一遍又一遍，当然不至于泪流满面从此改变人生之类的鬼话，但真的特别喜欢以至于这些视频不管移动硬盘怎么腾挪转移都不可能删除的。还把喜欢的歌无损版、320kbps版都下载下来为了在电脑上和MP3里各一份。不过这些年早就不怎么听歌了，家里每天充斥的也是“一只哈巴狗爱吃肉骨头…”之类的儿歌或者“天猫精灵，播放奥特曼主题曲”。当年的移动硬盘不知道在哪吃灰，也不知道还能不能读取。再看到这本小册子时又勾起了年轻的回忆，这回确实有些许感动。推荐每一位年轻或者不再年轻的朋友找到live aid的歌单听一听，听过这些猛的音乐，再听那些平庸的音乐就觉得寡淡无味了。

  • 司马迁的记忆之野 中国青春三部曲之三，借着司马迁的记忆聊聊汉武帝时期的一些事儿。

  • 呐，妈妈 一本漫画，我是看不太懂漫画。

  • 匏瓜 : 读《史记·孔子世家》 刘勃老师真是高产，以《孔子世家》为主结合其他史料所写的孔子传记，还原一个尽量真实且具有人性的圣人。

  • 医学大神 这套书是现代医学封神榜，讲述14位现代医学大神的生平和他们对现代医学的贡献。既有弗莱明发现青霉素，詹纳通过接种牛痘从而消灭天花病毒，巴斯德发现细菌致病从而发明人工疫苗，南丁格尔开创现代护理学，等等这些耳熟能详的故事。还有很多可能没听过但他们同样在现代医学发展中有开创卓越传奇的成就，比如真正打开现代医学大门的解剖学大师维萨里，《人体结构》出版的时候我国正处大明王朝嘉靖时代；建立血液循环理论，并把实验方法引入医学的哈维；建立产褥热理论的塞麦尔维斯，挽救了无数产妇的生命；人品略差胆子大的“老赖”牙医莫顿把乙醚用于麻醉，让刚拔牙的我深感伟大；“教授中的教授”菲尔绍是现代病理学的开山始祖；李斯特推广手术消毒术让败血症并不那么高发，外科手术成为一门科学。细菌学家科赫归纳的鉴定鉴别细菌基本法则“科赫法则”至今依然不过时；现代临床医学之父奥斯勒；研发出606的埃尔利希拯救了数百万梅毒患者，发明的系统筛选法现在仍然是药物生产的核心技术；白求恩同志的同学班廷主导提纯的胰岛素又是一个拯救数百万人生命的药；说到青霉素大家都知道弗莱明的传奇故事，实际上能提纯青霉素，让青霉素真正成为救人最多的药物，弗洛里领导的牛津团队贡献更大…. 14个传奇故事14座医学丰碑。看完这套书，我更确信一个观点，文明进步基本是靠这极少部分天才推动的，我等芸芸众生最多在车轮后面喊了一句“牛逼”。传记故事少不了演绎，但总是恰到好处。作者先从医后从IT再从文，有专业背景也有理工科的严谨还有不错的文笔，所谓有趣有料。再说书，每一本150-200页，两三次通勤看完一本。轻薄的小册子即使是拿在手上盘一盘都别有一番风味。看完往就丢在书架，等着孩子长大去发现，这些故事绝对可以成为中学高分作文的绝妙素材。

其他 另外一些没有不知道怎么分主题的书

  • 我等就来唱山歌 那年乐夏听到九连真人的《莫欺少年穷》无比惊艳，作为客家人，虽然我们说的客家话与歌中所唱的客家话并不一样，但还是颇有代入感。后面知道了同样唱客家话的林生祥，交工乐队。反复听了一段时间更是无法自拔，客家人勤劳坚韧的性格在他们的歌里都有体现。看罢此书更是震惊于钟永丰的才华，最重要的是钟永丰是一位基层官员，这跨界可真是大。

  • 把自己作为方法 : 与项飙谈话 又是2020年大火的一本书，感觉读书群人人都夸过。人类学我不懂，但能感受到项飙的诚恳，实事求是做研究。

  • 金叶 : 来自金枝的故事 本书是一部严肃的研究原始信仰和巫术活动的科学著作，是一部在世界范围内研究古老习俗极其有关信仰、观念的科学巨著，是闻名遐迩的学术必读著作，由于该书搜集了丰富的人类学资料，被称为人类学的百科全书。以上来自《金枝》条目的内容简介。但实在看不进浩瀚的《金枝》，而这本《金叶》则是金枝作者老婆去枝存叶的小读本，感觉更像是一本青少年读本。但本书中光怪陆离的传说故事还是非常有趣。

  • 山中最后一季 : 一个将生命、灵魂与激情融入山野的故事 这是一本舍不得读完的书，穿插主人公蓝迪的故事和搜寻蓝迪的故事，两条线交叉前行。蓝迪的故事让人惊叹世界上存在一种人对大自然有宗教式的信仰，而搜寻蓝迪的故事却总让人揪心。

  • 夏日走过山间 这是一本可以让人内心平静的书，通过优美的文字去畅游大自然，领略大自然的美。可能译者优美的文笔得占一半功劳，感觉可以列为中学生必读课外书。

  • 置身事内 : 中国政府与经济发展 治大国如烹小鲜，水多加面面多加水。虽然有一些政治话术一些看似根据结论推导过程的马后炮，但这书还是非常值得阅读，普通人了解社会主义特色市场经济的运行机制的科普好书。这是2021年身边最多人评为年度最佳的书。

  • 别想摆脱书 : 艾柯、卡里埃尔对话录 两位大神有的没的侃大山，谈话总是少不了金句。
